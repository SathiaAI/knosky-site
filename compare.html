Positioning

Not another code brain.
A navigation layer.

Use Claude, Cursor, Codex, Hermes for authoring. Use KnoSky so they stop getting lost — and so teams can share routes and optional receipts.

KnoSky is

  • Local map of repos & docs
  • Shared GPS routes for agents
  • Optional allow / deny receipts
  • Packs that ride with your tools

KnoSky is not

  • A coding model or IDE replacement
  • A Sourcegraph/OpenWiki clone
  • Always-on multi-tenant SaaS of your code
  • A promise of automatic “swarm-safe” everywhere

PR neighbors stay honest
